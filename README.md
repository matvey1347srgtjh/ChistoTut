# Чисто.Тут

Система учёта заказов для химчистки: от приёма вещи до выдачи клиенту, с автоматическим расчётом скидки за лояльность.

---

## Стек

- **Backend:** Python, Django
- **Frontend:** Django Templates
- **БД:** SQLite

---

## Запуск проекта

1. **Создать и активировать виртуальное окружение**

   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

2. **Установить зависимости**

   ```bash
   pip install -r requirments.txt
   ```

3. **Перейти в папку проекта**

   ```bash
   cd ChistoTut
   ```

4. **Применить миграции** (если ещё не применялись)

   ```bash
   py manage.py migrate
   ```

5. **Запустить сервер**
   ```bash
   py manage.py runserver
   ```

Приложение будет доступно по адресу: **http://127.0.0.1:8000/**

---

## Первый запуск

Создать учётную запись администратора:

```bash
py manage.py createsuperuser
```

Вход — **http://127.0.0.1:8000/auth/login/**  
Регистрация сотрудника — **http://127.0.0.1:8000/auth/register/**
